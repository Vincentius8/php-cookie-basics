# PHP Cookie Basics (Phase 1)
Author: John Vincent Pangilinan

This repo contains my exercises for learning **cookies in PHP** — how to create, retrieve, and delete cookies.

## Activities
1. setCookie.php – Sets a cookie with a username or user preference
2. getCookie.php – Retrieves and displays the stored cookie value
3. deleteCookie.php – Removes an existing cookie
4. cookieForm.php – Simple form that saves user input into a cookie

## Purpose
To understand how cookies work in PHP for storing user data across browser sessions.
